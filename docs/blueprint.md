# **App Name**: MassMailer Pro

## Core Features:

- Importación de Datos: Importar datos desde archivos de Excel (.xlsx o .csv) que contengan información del destinatario y de la factura.
- Procesamiento y Formateo de Datos: Procesar y formatear los datos importados para crear información estructurada para la composición del correo electrónico. Hacer coincidir las facturas con los destinatarios y acumular cada factura para un proveedor.
- Herramienta de Composición de Correo Electrónico: Generar cuerpos de correo electrónico que incluyan información del destinatario y una tabla que contenga los detalles de la factura, estructurados para mayor claridad y presentación profesional.
- Integración con Outlook: Preparar el contenido del correo electrónico para una transferencia perfecta a Outlook, facilitando el proceso final de revisión y envío masivo por parte del usuario.

## Style Guidelines:

- Color primario: Azul suave (#64B5F6) para inspirar confianza y profesionalismo.
- Color de fondo: Gris claro (#F5F5F5) para una interfaz limpia y ordenada.
- Color de acento: Teal (#4DB6AC) para elementos interactivos y resaltados para brindar una sensación de eficiencia.
- Fuente del cuerpo: 'PT Sans', una fuente sans-serif humanista que proporciona un equilibrio entre modernidad y legibilidad, muy adecuada tanto para el texto en pantalla como para la legibilidad de los documentos.
- Fuente del encabezado: Fuente sans-serif 'Space Grotesk' para darle una sensación informática y tecnológica.
- Utilice iconos sencillos y claros para representar acciones y tipos de datos dentro de la aplicación.
- Priorice un diseño claro y lógico que guíe al usuario a través del proceso de importación de datos, composición de correo electrónico y exportación para Outlook.
